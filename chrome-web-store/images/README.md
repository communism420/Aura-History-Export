# Chrome Web Store Images

Prepared image assets:

- `store-icon-128.png` - required extension icon, 128x128 PNG.
- `small-promo-440x280.png` - required small promotional tile, 440x280 PNG.
- `screenshot-export-1280x800.png` - required-size screenshot, 1280x800 PNG.
- `screenshot-beginner-1280x800.png` - required-size screenshot, 1280x800 PNG.
- `screenshot-import-archive-1280x800.png` - required-size screenshot, 1280x800 PNG.
- `screenshot-privacy-1280x800.png` - required-size screenshot, 1280x800 PNG.
- `large-promo-920x680.png` - optional large promotional tile, 920x680 PNG.
- `marquee-promo-1400x560.png` - optional marquee promotional tile, 1400x560 PNG.
- `very-large-promo-1400x560.png` - use this for the Chrome Web Store "Very large promotional image" slot, 1400x560 24-bit PNG without alpha.
- `ru/` - localized Russian screenshots for the Russian Chrome Web Store listing.

The screenshots are representative Chrome Web Store screenshots of the extension interface and features. The required extension icon is also included inside the release ZIP through `assets/icon-128.png`.
