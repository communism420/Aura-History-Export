# Chrome Web Store Listing: Aura History Export

Use this copy in the Chrome Web Store Developer Dashboard. Keep the privacy policy, dashboard disclosures, and actual extension behavior consistent.

## English

### Name

Aura History Export

### Short Description

Fully open-source Chromium browser history export to different popular formats

### Detailed Description

Export your available Chromium browser history to different popular formats. Choose a date range, select the columns you need, preview records, filter domains and protocols, import Google Takeout history files, and optionally keep a local archive for future exports.

Aura History Export is fully open-source. The full source code is public, so anyone can inspect how history is handled and verify that processing stays local.

All processing happens locally in your browser. Aura History Export does not use analytics, ads, tracking, remote code, accounts, or external servers. Your history is not sent anywhere.

## Русский

### Название

Aura History Export

### Краткое описание

Полностью open-source экспорт истории Chromium-браузера в популярные локальные форматы.

### Полное описание

Aura History Export помогает экспортировать историю браузера из Chromium-браузеров в удобные локальные файлы.

Проект полностью open-source: исходный код публичен, поэтому любой может проверить, как расширение обрабатывает историю и что обработка остается локальной.

Основные возможности:

- Экспорт всех локальных посещений, доступных через Chromium History API.
- Выбор периода: последние 24 часа, 7 дней, 30 дней, 90 дней, вся доступная история или свои даты.
- Выбор столбцов: время посещения, название страницы, домен, URL, количество посещений, ручной ввод, тип перехода, ID визита-источника и источник данных.
- Выбор типа файла прямо в интерфейсе расширения при создании экспорта.
- Режим новичка с упрощенным экспортом.
- Экспертный режим с фильтрами, предпросмотром, группировкой доменов, импортом Google Takeout и опциональным локальным архивом.
- Импорт поддерживаемых файлов истории Google Takeout.
- Опциональный локальный архив для будущих экспортов.

Приватность:

- Вся обработка выполняется локально в браузере.
- История не отправляется разработчику или на сторонние серверы.
- Нет аналитики, телеметрии, трекинга, рекламы, удаленного кода и входа в аккаунт.
- Разрешения ограничены историей браузера и локальным хранилищем расширения.

Ограничение Chromium:

Расширение экспортирует историю, доступную локальному профилю Chromium через History API. Для более старых данных используйте импорт Google Takeout или заранее включите локальный архив.
